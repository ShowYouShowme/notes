## 安装python3

```shell
yum install python36 -y
```



## 安装pip

```shell
# step1
curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py

# step2
python3 ./get-pip.py
```



## 安装PyMySQL

1. 安装

   ```shell
   pip install PyMySQL
   ```

2. PyMySQL与python2.x的同时存在，PyMySQL被安装到了python2.7

   ```shell
   cp /usr/lib/python2.7/site-packages/pymysql/ /usr/lib/python3.6/site-packages
   ```

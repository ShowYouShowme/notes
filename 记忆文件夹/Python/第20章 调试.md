# 调试

## pdb

| 命令                    | 简写                | 作用                   |
| ----------------------- | ------------------- | ---------------------- |
| break                   | b                   | 设置断点               |
| continue                | c                   | 继续执行               |
| list                    | l                   | 查看当前行代码段       |
| **step**                | **s**               | **进入函数**           |
| return                  | r                   | 从当前函数返回         |
| quit                    | q                   | 终止并退出             |
| **next**                | **n**               | **执行下一行**         |
| **print**               | **p**               | **打印变量值**         |
| help                    | h                   | 帮助信息               |
| args                    | a                   | 查看传入参数           |
| break                   | b                   | 显示所有断点           |
| break ${lineno}         | b ${lineno}         | 在指定行设置断点       |
| break ${file}:${lineno} | b ${file}:${lineno} | 在指定文件的行设置断点 |
| clear ${num}            |                     | 删除指定断点           |
| bt                      |                     | 查看函数调用堆栈       |



## 调试命令

```shell
python3 -m pdb some.py
```

### 介绍和安装

+ 语言特点
	1. 语法简洁
	2. 跨平台
	3. 类库丰富
	4. 开放源码
	5. 可扩展，可以调用C/C++的库

	+ 发展历史

		1. Python2.x

			1. 官方版本
			2. 发行版本[Anaconda](https://www.anaconda.com/)

		2. [iPython](https://ipython.org/)

		4. sublime text 文本编辑器
		5. Pycharm 集成开发环境
		6. Pip 包管理器

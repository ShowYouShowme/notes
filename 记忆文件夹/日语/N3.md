# 第一课

1. 动词辞书形 +  一方だ：一直……

   ```
   仕事を辞めてから、私の貯金は減る一方です。
   忙しくて、運動する時間もないから太る一方だ。
   ```

2. 普通形（な・の）うえに：不但A而且B

   ```
   この店の刺身は安いうえに、美味しい。
   
   この店の刺身は高いうえに、美味しくない。
   
   彼の話は長いうえに、つまらない。
   ```

3. 普通形（な・の）うちに：趁A~

   ```
   雨が降らないうちに、早くうちに帰ろう。
   
   熱いうちに召し上がってください。
   ```

4. 普通形（な・の）おかげで：托A的福

   ```
   先生のおかげで、試験に合格でくました。
   
   おかげさまで、体の調子も良くなりました。
   ```

5. 〜おそれがある：恐怕A

   ```
   动词 辞书形 +　おそれがある
   ない形 　　　+ おそれがある
   名词 の　　　+ おそれがある 
   ```

6. 〜かける・〜かけ

   ```
   动词ます（去掉）　かけて
   　　　　　      かける    刚开始做
   　　　　　      
   かけの＋名词
   かけだ＋
   
   机の上には食べかけのカレーライスの皿が置いてあった。
   ```

7. 〜がち：经常A，有A的倾向

   ```
   动词ます形 がち
   名词 がち
   
   忘れがち　＝＞よく忘れる
   病気がち　＝＞　よく病気になる
   ```

8. 名词から名词にかけて：从A到B

   ```
   五月から八月nにかけて、美味しいマンゴーを味わうことができます。
   
   朝７時から８時半にかけて電車は大変込みます。
   ```

9. 〜からには：既然A

   ```
   普通形（） + からには
   
   ベトナムに転勤するからには、現地の言葉が話せるように頑張りたい。
   ```

10. 〜から見て：从A来看

   ```
   名词 + から見て
   　　　　から見ると
   　　　　から見れば
   　　　　から見たら
   コストの面から見ると、実現が難しい。
   
   大きさから見て、足跡は熊のものだと思います。
   ```

11. 〜かわりに：代替，另一方面，代价

    ```
    普通形（な・の）かわりに
    
    今日は田中先生のかわりに私が授業します。
    ここは家賃が安いかわりに、交通が不便だ。
    引っ越しを手伝ってもらったかわりに、食事をごちそします。
    ```

12. 〜きる・〜きれる：做完，做到极限，做到最后

    ```
    动词ます形（去掉ます）　＋　きる
    　　　　　　　　　　　　＋　きれる
    食べきる　＝＞　全部食べる
    食べきれる　＝＞　全部食べられる
    
    こんなたくさんのご馳走は一人では食べきれません。
    ```

13. 〜くらい・くらいだ：像A的程度，像A那样

    ```
    普通形（な）くらい
    　　　　　　くらいだ
    　　　　　　
    口から火が出るほど辛い。
    口から火が出るくらい辛い。
    
    死ぬほど疲れだ。
    
    床下には大人三人が入れるくらいのスペースがあった。
    隣の部屋がうるさくて、今すぐ引っ越したいぐらいだ。
    ```

14. 〜こそ：才是、正是

    ```
    あなたこそうちの会社に必要な人材だ。
    
    今年こそ新しい彼氏を作りたい。
    ```

15. 〜ことから・〜ところから：因为

    ```
    普通形（）ことから
    　　　　　ところから
    　　　　　名词から
    　　　　　
    彼は太っていることから、相撲部に入らせられました。
    たばこのひの不始末から、火事になった。
    ```



# 第二课

1. 〜のことだから・〜のことだ：因为是

   ```
   名词のことだから
      のことだ
      
   あれ？田中さんがいませんね。
   （時間にルーズな）田中さんのことだから。今日も遅刻だろう。
   
   雄太のことだ。まだ宿題をやっていないに違いない。
   ```

2. 〜ことはない：没必要，用不着

   ```
   动词辞书形 + ことはない
   
   飛行機が揺れるおはよくあることだから。心配することはないよ。
   ```

3. 〜最中に：正在……的时候，正好是……

   ```
   动词て形いる +　最中に
   名词 の+　最中だ
   
   人が話している最中に居眠りを始めるなんて失礼すぎます。
   ```

4. 〜際に・〜際は：……的时候

   ```
   动词普通形 + 際「に」「は」
   名词の　際「に」「は」
   
   与時・場合类似
   
   火事の際はエレベーターを使わないで、階段を使って避難してください。
   
   降りる際は足元にご注意ください。
   ```

5. 〜ことになっている：按照规定……，有决定……

   ```
   动词辞书形/ない形　ことになっている
   
   今晩６時に西村さんと田中さんで会うことになっている。
   
   来月から日系企業で日本語を教えることになってうる。
   ```

6. 〜ことにしている：决定……

   ```
   动词辞书形/ない形　ことにしている
   
   午後３時を過ぎたら、コーヒーを飲まないことにしている。
   
   他人の悪口は絶対に言わないことにしています。
   ```

7. 〜ことになる：也就是说……

   ```
   动词普通形ことになる
   
   一日１００グラム痩せれば、一か月で３キロ痩せることになる。
   
   社長が急用で出社できないので、今日の会議は延期ということになります。
   ```

8. 〜さえ：连……都，甚至……

   ```
   名词[助词]さえ
   　　　でさえ
   　　　
   彼は三日間、水さえ飲まなかった。
   子供でさ交通ルールを守っている。
   この事は親にさえ話していない。
   ```

9. 〜さえ〜ば：只要……就

   ```
   値段が安くさえあれば、私は買います。
   交通が便利でさえあれば、そこに引っ越します。
   
   あなたさえいてくれれば、他には何も入りません。
   ```

10. 〜しかない：只好……，只能……

    ```
    动词辞书形 + しかない
    动作性名词
    終電もなくなったので、タクシーで帰るしかない。
    ここまできたら、最後まで頑張るしかありません。
    ```

11. 〜せい：因为……，都怪……

    ```
    普通型（な・の）せいで
    　　　　　　　　せいか
    　　　　　　　　せいだ
    あの医者のせいで、命を落とした。
    夜も暑いせいで、よく眠れない。
    ```

12. 〜だけ・だけの：……多少，尽量……

    ```
    今日はできるだけ早く帰りたいです。
    好きなだけ食べてもいいです。
    
    ここにあるものは持てるだけ持って行ってください。
    ```

13. 〜たところ：……结果

    ```
    动词た形　＋　ところところ
    
    Youtubeで日本語を勉強していたところ、出口先生の動画を見つけました。
    ```

14. 〜たとたん：刚……就

    ```
    动词た形　たとたん
    
    部屋の窓を開けたとたん、買っていた小鳥が飛んでいってしまた。
    
    家を出たとたん、雨が降ってきた。
    ```

15. 〜たびに：每次……，每当……

    ```
    动词 辞书形 ＋　たびに
    名词の
    
    おばあちゃんは会うたびに、お菓子をくれます。
    工藤さんは出張のたびに、私たちにお土産を買ってきてくれる。
    ```



# 第三课

1. 〜だらけ：全是……，满是……

   ```
   名词だらけ
   
   彼の部屋は掃除や片付けをしていないようで、ゴミだらけでした。
   うちの職場は男性だらけで、出会いの機会がない。
   ```

2. 〜ついでに：顺便

   ```shell
   动词辞书形/た形 + ついでに
   动作性名词の
   
   友人を駅まで送ったついでに、スーパーで夜食を買ってきた。
   子供を車で駅まで送ったついでに、図書館で本を借りてきた。
   
   ```

3. 〜っけ：……来着，回顾

   ```
   今日は何曜日だっけ？
   佐藤さんは独身でしたっけ？
   学生の頃、よく彼女とこの喫茶店に来たっけ
   
   ```

4. 〜っぽい：容易……，有点……，好像

   ```
   动词去ます形 + っぽい
   名词
   
   うちの上司は怒りっぽい。
   
   ちょっと風邪っぽい。
   
   彼は考え方が子供っぽい。
   ```

5. 〜てからでないと・〜てからでなければ：若不先……就

   ```
   动词て形 + からでないと
   　　　　　　からでなければ
   　　　　　　
   準備運動をしてからでなければ、怪我をするリスクが高い。
   実際に部屋を見てからでなければ、借りるかどうか決められません。
   ```

6. 〜てしょうがない・〜てしかたがない：非常……，……得不得了

   ```
   动词て形　　　+ しょうがない
   い形容词くて
   な形容词で
   
   好きな人と付き合えることになって、嬉しくてしょうがない。
   クーラーが壊れてしまって、暑くてしょうがない。
   
   眠くてしょうがないから、ちょっと横にならせてください。
   ```

7. 〜てたまらない：……得不得了

   ```
   动词て形　　　+ たまらない
   い形容词くて
   な形容词で
   
   留学生活を始めたばかりの頃は、家族に会いたくてたまらなかった。
   受験勉強から解放されて、兄は大学生活が楽しくてたまらないようです。
   ```

8. 〜ということだ・〜とのことだ：据说，听说

   ```
   A社からの回答は、これ以上の値下げは難しいということです。
   ```

9. 〜ということだ：也就是

   ```
   会場には駐車場が全くありません。つまり車では来るなということだ。
   
   明日から会社に来ないていい？つまり私はクビということですか？
   ```

10. 〜というより：与其……不如

    ```
    　　　　な形容词/名词
    普通形（だ）          というより
    
    現在の気温はマイナス１０度です、屋外に出ると、寒いというより寒いです。
    私は健康のためというより、ストレス発散のためにジョギングをしている。
    ```

11. 〜なくちゃ・〜なちゃ・〜ないと：必须……，不……不行

    ```
    ない形　なくちゃ
    　　　　なちゃ
    　　　　ないと
    こんな時間だ。もうかえらないと
    明日も早いから、もう寝なきゃ。
    ```

12. 指示词

    ```
    そ系列：只有一方知道
    あ系列：双方都知道
    
    去年会社を辞めた村田さん、今あの人、何をしているでしょうか。
    
    前回待ち合わせたあの場所、覚えてる？あそこでまた会おう。
    ```

13. 〜といっても：虽说……

    ```
          な形容词/名词
    普通形（だ）　といっても
    車を買ったといっても、中古の安い車です。
    
    家を買ったといっても、郊外の中古のマンションです。
    ```

14. 〜とおりに：按照……，正如……

    ```
    动词辞书形/た形　とおりに
    名词の
    
    今から私が説明するとおりにパソコンを操作してください。
    レシピのとおりに作りましたが、あまり美味しくないです。
    ```

15. 〜ところへ・〜ところに・〜ところで・〜ところを：……的时候

    ```
    辞书形　　　ところへ
    て形いる　　ところに
    た形　　　　ところで
    い形容词　　ところを
    
    彼女と楽しくおしゃべりしているところへ苦手な久保さんがやってきた。
    40ページを教え終わったところで、授業終了のチャイムが鳴りました。
    ```



# 第四课

1. 〜としたら・〜とすれば・〜とすると：如果

   〜としても：即使……也

   ```
    もし100万円もらえるとしたら、何を買いますか
    もし100万円もらえるとしても、そんな仕事はしたくない。
    
    高い給料をもらえるとしても、その仕事はしたくないです。
   ```

2. 〜として：作为……，身为……

   ```
   名词として
   
   私は筑波大学の代表として、このシンポジウムに参加しました。
   
   この国では、オートバイが通勤手段として、広く普及しています。
   ```

3. 〜など・〜なんか・〜なんて：表示举例；各种强调

   ```
   举例
   ノートや鉛筆などを買いました。
   
   强调
   お前など大嫌いだ。
   ```

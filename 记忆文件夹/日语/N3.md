# 第一课

1. 动词辞书形 +  一方だ：一直……

   ```
   仕事を辞めてから、私の貯金は減る一方です。
   忙しくて、運動する時間もないから太る一方だ。
   ```

2. 普通形（な・の）うえに：不但A而且B

   ```
   この店の刺身は安いうえに、美味しい。
   
   この店の刺身は高いうえに、美味しくない。
   
   彼の話は長いうえに、つまらない。
   ```

3. 普通形（な・の）うちに：趁A~

   ```
   雨が降らないうちに、早くうちに帰ろう。
   
   熱いうちに召し上がってください。
   ```

4. 普通形（な・の）おかげで：托A的福

   ```
   先生のおかげで、試験に合格でくました。
   
   おかげさまで、体の調子も良くなりました。
   ```

5. 〜おそれがある：恐怕A

   ```
   动词 辞书形 +　おそれがある
   ない形 　　　+ おそれがある
   名词 の　　　+ おそれがある 
   ```

6. 〜かける・〜かけ

   ```
   动词ます（去掉）　かけて
   　　　　　      かける    刚开始做
   　　　　　      
   かけの＋名词
   かけだ＋
   
   机の上には食べかけのカレーライスの皿が置いてあった。
   ```

7. 〜がち：经常A，有A的倾向

   ```
   动词ます形 がち
   名词 がち
   
   忘れがち　＝＞よく忘れる
   病気がち　＝＞　よく病気になる
   ```

8. 名词から名词にかけて：从A到B

   ```
   五月から八月nにかけて、美味しいマンゴーを味わうことができます。
   
   朝７時から８時半にかけて電車は大変込みます。
   ```

9. 〜からには：既然A

   ```
   普通形（） + からには
   
   ベトナムに転勤するからには、現地の言葉が話せるように頑張りたい。
   ```

10. 〜から見て：从A来看

   ```
   名词 + から見て
   　　　　から見ると
   　　　　から見れば
   　　　　から見たら
   コストの面から見ると、実現が難しい。
   
   大きさから見て、足跡は熊のものだと思います。
   ```

11. 〜かわりに：代替，另一方面，代价

    ```
    普通形（な・の）かわりに
    
    今日は田中先生のかわりに私が授業します。
    ここは家賃が安いかわりに、交通が不便だ。
    引っ越しを手伝ってもらったかわりに、食事をごちそします。
    ```

12. 〜きる・〜きれる：做完，做到极限，做到最后

    ```
    动词ます形（去掉ます）　＋　きる
    　　　　　　　　　　　　＋　きれる
    食べきる　＝＞　全部食べる
    食べきれる　＝＞　全部食べられる
    
    こんなたくさんのご馳走は一人では食べきれません。
    ```

13. 〜くらい・くらいだ：像A的程度，像A那样

    ```
    普通形（な）くらい
    　　　　　　くらいだ
    　　　　　　
    口から火が出るほど辛い。
    口から火が出るくらい辛い。
    
    死ぬほど疲れだ。
    
    床下には大人三人が入れるくらいのスペースがあった。
    隣の部屋がうるさくて、今すぐ引っ越したいぐらいだ。
    ```

14. 〜こそ：才是、正是

    ```
    あなたこそうちの会社に必要な人材だ。
    
    今年こそ新しい彼氏を作りたい。
    ```

15. 〜ことから・〜ところから：因为

    ```
    普通形（）ことから
    　　　　　ところから
    　　　　　名词から
    　　　　　
    彼は太っていることから、相撲部に入らせられました。
    たばこのひの不始末から、火事になった。
    ```

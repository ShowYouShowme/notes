# 第四课



## 4.1 文型

1. 今四時五分です
2. 私は九時から五時まで働きます
3. 私は朝六時に起きます
4. 私は昨日勉強しました



## 4.2 语法

1. 动词的分类

   + 一类动词：词尾不是る；ア、ウ、オ段 + る

     ```
     １、休む
     ２、働く
     ３、終わる
     ４、分かる
     
     
     特殊一类动词
     １、帰る
     ２、切る
     ３、知る
     ４、入る
     ５、いる
     ```

     

   + 二类动词：ｲ、エ段+る

     ```
     １、起きる
     ２、寝る
     ```

     

   + 三类动词：する、くる

     ```
     １、勉強する
     ２、来る
     ```

     

     

2. 动词句

   动词连用形 + ます

   |      | 现在、将来 | 过去         |
   | ---- | ---------- | ------------ |
   | 肯定 | ます       | ました       |
   | 否定 | ません     | ませんでした |

   

3. 时间表示

   ```ini
   １、小时
   4点 = よじ
   7点 = しちじ
   ９点 = くじ
   
   2、分钟
   1分 = 　いっぷん
   3分 =　さんぷん
   4分 = よんぷん
   6分 = ろっぷん
   8分 = 　はっぷん
   10分=　じゅっぷん
   30分= はん
   ```

   

4. 在某个时间做某事

   ```
   １、具体时间  に + 动词
   十一時に起き
   
   ２、非具体时间
   私は毎日働きます
   ```

   

5. から・・・まで・・・

   ```
   １、九時から五時まで、
   ２、十時から勉強します
   ３、九時まで寝ました
   ```

   

6. と：相当于和

   ```
   休みは土曜日と日曜日です
   ```

   

# 第五课



## 5.1 文型

1. 私は京都へ行きます
2. 私はタクシーで家へ帰ります
3. 私は家族と日本へきました





## 5.2 语法

1. へ表示动作、作用的方向

   ```
   うちへ帰ります
   ```

2. も表示全面否定

   ```
   １、誰も知りません
   ２、どこも行きません
   ```

3. で表示手段、方法。

   ```
   １、電車で行きます
   ２、何でここへきましたか
   ```

4. と表示共同做某事的对象，和

   ```
   家族と日本へ来ました
   ```

   



# 第六课



## 6.1 文型

1. 私はジュースを飲みます
2. 私は駅で新聞を買います
3. いっしょにこうべへ行きまえんか
4. ちょっと休みましょう



## 6.2 语法

1. を提示宾语

   ```
   私は肉を食べます
   ```

2. する做某事

   ```
   １、运动
   サッカーをします
   
   2、集会
   パーティーをします
   
   3、其他
   宿題をします
   ```

3. 询问做什么

   ```
   月曜日何をしますか。
   京都へ行きます
   ```

4. で表示行为发生的场所

   ```
   駅で新聞を買います
   ```

5. 表示邀请的两种说法

   ```
   1、ませんか
   一緒に京都へ行き都へ行きませんか
   
   ２、ましょう
   一緒に飲みましょう
   ```

# 第八课



## 8.1 文型

1. 桜はきれいです
2. 富士山は高いです
3. 桜は綺麗い花です
4. 富士山はたかい山です



## 8.2 语法

1. 形容词的分类

   | 分类         | 例子                         |
   | ------------ | ---------------------------- |
   | い形容词     | いい、美味しい、古い、小さい |
   | な形容词     | ハンサム、簡単、元気         |
   | 特殊な形容词 | 綺麗い、嫌い                 |

2. 形容词做定语

   ```
   １、い形容词 + 名词
   サントスさんはいい人です。
   
   2、な形容词 + な + 名词
   京都は有名な町です
   ```

   

3. 形容词做谓语

   |      | い形容词               | な形容词               |
   | ---- | ---------------------- | ---------------------- |
   | 肯定 | 〜　＋　です           | 〜　＋　です　簡単です |
   | 否定 | 〜い　ー＞くない＋です | 〜　＋　ではありません |

   ```
   肯定
   1、安いです
   ２、簡単です
   
   否定
   １、安くないです
   ２、簡単ではありません
   
   
   注意：いい否定形式是形式是よくない　
   ```

   

4. どんな　什么样的

   ```
   奈良はどんな町ですか
   古いまちです
   ```

   

5. が　但是

   ```
   日本の食べ物は美味しいですが、高いです
   ```

   

6. そして：而且

   ```
   Tokyoの地下鉄は綺麗いです。そして、便利です。
   ```

   

7. どう：怎么样

   ```
   日本の生活はどうですか
   ```

   

8. とても・あまり　程度副词，修饰形容词时放在形容词前面

   ```
   1、 とても　+ 肯定
   北京はとても寒いです
   これはとても有名な映画です
   
   2、 あまり　+ 否定
   シャンハイはあまり寒くないです
   さくら大学はあまり有名な大学じゃありません
   ```

   

9. 哪个：从三件物品以上，选出一个特定的。

   ```
   ミラーさんの傘はどれですか
   ```



# 第九课



## 9.1 文型

1. 私はイタリア料理が好きです
2. 私は日本語が少しわかります
3. 今日は子供の誕生日ですから、早く帰ります。



## 9.2 语法

1. が表示情感、能力的对象

   ```
   私は歌が下手です
   ```

2. 副词

   ```
   １、よく
   ２、だいたい
   ３、たくさん
   ４、少し
   ５、あまり
   ６、全然
   
   
   11、表示程度
   よくわかります
   だいたいわかります
   少しわかります
   
   あまりわかりません
   全然わかりません
   
   12、表示数量
   たくさんあります
   少しあります
   あまりありません
   全然ありません
   ```

   

3. から表示因果关系

   ```
   時間がありませんから、新聞をの読みません
   ```

4. どうして询问原因。回答时句尾加から

   ```
   どうして新聞をの読みませんか
   時間がありませんから
   ```

5. どんな　＋　名词，询问某个范围内的具体名称

   ```
   どんなスポーツが好きですか
   野球が好きです
   ```

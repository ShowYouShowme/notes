# 第四课



## 4.1 文型

1. 今四時五分です
2. 私は九時から五時まで働きます
3. 私は朝六時に起きます
4. 私は昨日勉強しました



## 4.2 语法

1. 动词的分类

   + 一类动词：词尾不是る；ア、ウ、オ段 + る

     ```
     １、休む
     ２、働く
     ３、終わる
     ４、分かる
     
     
     特殊一类动词
     １、帰る
     ２、切る
     ３、知る
     ４、入る
     ５、いる
     ```

     

   + 二类动词：ｲ、エ段+る

     ```
     １、起きる
     ２、寝る
     ```

     

   + 三类动词：する、くる

     ```
     １、勉強する
     ２、来る
     ```

     

     

2. 动词句

   动词连用形 + ます

   |      | 现在、将来 | 过去         |
   | ---- | ---------- | ------------ |
   | 肯定 | ます       | ました       |
   | 否定 | ません     | ませんでした |

   

3. 时间表示

   ```ini
   １、小时
   4点 = よじ
   7点 = しちじ
   ９点 = くじ
   
   2、分钟
   1分 = 　いっぷん
   3分 =　さんぷん
   4分 = よんぷん
   6分 = ろっぷん
   8分 = 　はっぷん
   10分=　じゅっぷん
   30分= はん
   ```

   

4. 在某个时间做某事

   ```
   １、具体时间  に + 动词
   十一時に起き
   
   ２、非具体时间
   私は毎日働きます
   ```

   

5. から・・・まで・・・

   ```
   １、九時から五時まで、
   ２、十時から勉強します
   ３、九時まで寝ました
   ```

   

6. と：相当于和

   ```
   休みは土曜日と日曜日です
   ```

# 第八课



## 8.1 文型

1. 桜はきれいです
2. 富士山は高いです
3. 桜は綺麗な花です
4. 富士山はたかい山です



## 8.2 语法

1. 形容词的分类

   | 分类         | 例子                         |
   | ------------ | ---------------------------- |
   | い形容词     | いい、美味しい、古い、小さい |
   | な形容词     | ハンサム、簡単、元気         |
   | 特殊な形容词 | 綺麗、嫌い                   |

2. 形容词做定语

   ```
   １、い形容词 + 名词
   サントスさんはいい人です。
   
   2、な形容词 + な + 名词
   京都は有名な町です
   ```

   

3. 形容词做谓语

   |      | い形容词               | な形容词               |
   | ---- | ---------------------- | ---------------------- |
   | 肯定 | 〜　＋　です           | 〜　＋　です　簡単です |
   | 否定 | 〜い　ー＞くない＋です | 〜　＋　ではありません |

   ```
   肯定
   1、安いです
   ２、簡単です
   
   否定
   １、安くないです
   ２、簡単ではありません
   
   
   注意：いい否定形式是形式是よくない　
   ```

   

4. どんな　什么样的

   ```
   奈良はどんな町ですか
   古いまちです
   ```

   

5. が　但是

   ```
   日本の食べ物は美味しいですが、高いです
   ```

   

6. そして：而且

   ```
   東京の地下鉄は綺麗です。そして、便利です。
   ```

   

7. どう：怎么样

   ```
   日本の生活はどうですか
   ```

   

8. とても・あまり　程度副词，修饰形容词时放在形容词前面

   ```
   1、 とても　+ 肯定
   北京はとても寒いです
   これはとても有名な映画です
   
   2、 あまり　+ 否定
   シャンハイはあまり寒くないです
   さくら大学はあまり有名な大学じゃありません
   ```

   

9. 哪个：从三件物品以上，选出一个特定的。

   ```
   ミラーさんの傘はどれですか
   ```

4. 私は一年で夏が一番好きです

## 12.2 语法

1. 形容词和名词的过去式

   ```
   [い形容词]
   肯定    = 美味しいです
   否定    =美味しくないです
   过去肯定 = 美味しかったです
   过去否定 =美味しくなかったです
   
   [な形容词]
   肯定    = 静かです
   否定    =　静かではありません
   过去肯定 = 静かでした
   过去否定 =　静かではありませんでした
   
   [名词]
   肯定    = 雨です
   否定    =　雨ではありません
   过去肯定 = 雨でした
   过去否定 =　雨ではありませんでした
   
   [动词]
   肯定    = 働きます
   否定    =　働きません
   过去肯定 = 働きました
   过去否定 =　働きませんでした
   ```

   

2. より　...比...

   ```
   北海道は九州より大きいです
   ```

3. 哪个更....

   ```
   名词1と名词2　と　どちらが形容词ですか
   テニスとサッカーと、どちらが面白いです
   
   サッカーのほが面白いです
   ```

# 第十四课



## 14.1 文型

1. ちょっと待ってください
2. ミラーさんは今電話をかけています

## 14.2 语法

1. 动词的て形

   + 一类动词

     ```
     １、く　ー＞　いて　ぐ　ー＞　いで
     書くー＞　かいて　泳ぐー＞　およいで
     
     ２、う、つ、る结尾→去掉う、つ、る+って
     もらう　ー＞　もらって　待つ　ー＞　まって　ある　ー＞　あって
     
     ３、む、ぬ、ぶ结尾→去掉む、ぬ、ぶ＋んで
     読む　ー＞　よんで　死ぬ　ー＞　しんで　遊ぶ　ー＞　あそんで
     
     ４、す结尾→して
     貸す　ー＞　かして
     
     ５、特殊 いく　ー＞　行って
     ```

     

   + 二类动词

     ```
     去掉 る　+　て
     食べる　ー＞　食べて
     ```

     

   + 三类动词

     ```
     する　ー＞　して
     くる　ー＞　きて
     ```

3. 〜ています表示动作正在进行


4. 〜ましょうか提议为对方做某事，看到对方有困难。


   

5. が表示轻微的转折

   ```

   ```

   

6. が提示眼前发生的现象

   ```
   名词 が动词

   ```

7. 特殊

   ```
   喫茶店に入ります
   
   喫茶店を出ます

     ```

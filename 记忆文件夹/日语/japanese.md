# 第四课



## 4.1 文型

| 今四時五分です               |
| ---------------------------- |
| 私は九時から五時まで働きます |
| 私は朝六時に起きます         |
| 私は昨日勉強しました         |



## 4.2 语法

1. 动词的分类

   + 一类动词：词尾不是る；ア、ウ、オ段 + る

     ```
     １、休む
     ２、働く
     ３、終わる
     ４、分かる
     
     
     特殊一类动词
     １、帰る
     ２、切る
     ３、知る
     ４、入る
     ５、いる
     ```

     

   + 二类动词：ｲ、エ段+る

     ```
     １、起きる
     ２、寝る
     ```

     

   + 三类动词：する、くる

     ```
     １、勉強する
     ２、来る
     ```

     

     

2. 动词句

   动词连用形 + ます

   |      | 现在、将来 | 过去         |
   | ---- | ---------- | ------------ |
   | 肯定 | ます       | ました       |
   | 否定 | ません     | ませんでした |

   

3. 时间表示

   ```ini
   4点 = よじ
   7点 = しちじ
   ９点 = くじ
   ```

   

4. 在某个时间做某事

   ```
   １、具体时间  に + 动词
   十一時に起き
   
   ２、非具体时间
   私は毎日働きます
   ```

   

5. から・・・まで・・・

   ```
   １、九時から五時まで、
   ２、十時から勉強します
   ３、九時まで寝ました
   ```

   

6. と：相当于和

   ```
   休みは土曜日と日曜日です
   ```

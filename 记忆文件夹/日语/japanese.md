# 第四课



## 4.1 文型

1. 今四時五分です
2. 私は九時から五時まで働きます
3. 私は朝六時に起きます
4. 私は昨日勉強しました



## 4.2 语法

1. 动词的分类

   + 一类动词：词尾不是る；ア、ウ、オ段 + る

     ```
     １、休む
     ２、働く
     ３、終わる
     ４、分かる
     
     
     特殊一类动词
     １、帰る
     ２、切る
     ３、知る
     ４、入る
     ５、いる
     ```

     

   + 二类动词：ｲ、エ段+る

     ```
     １、起きる
     ２、寝る
     ```

     

   + 三类动词：する、くる

     ```
     １、勉強する
     ２、来る
     ```

     

     

2. 动词句

   动词连用形 + ます

   |      | 现在、将来 | 过去         |
   | ---- | ---------- | ------------ |
   | 肯定 | ます       | ました       |
   | 否定 | ません     | ませんでした |

   

3. 时间表示

   ```ini
   １、小时
   4点 = よじ
   7点 = しちじ
   ９点 = くじ
   
   2、分钟
   1分 = 　いっぷん
   3分 =　さんぷん
   4分 = よんぷん
   6分 = ろっぷん
   8分 = 　はっぷん
   10分=　じゅっぷん
   30分= はん
   ```

   

4. 在某个时间做某事

   ```
   １、具体时间  に + 动词
   十一時に起き
   
   ２、非具体时间
   私は毎日働きます
   ```

   

5. から・・・まで・・・

   ```
   １、九時から五時まで、
   ２、十時から勉強します
   ３、九時まで寝ました
   ```

   

6. と：相当于和

   ```
   休みは土曜日と日曜日です
   ```

## 5.2 语法

   ```
   うちへ帰ります
   ```

   ```
   １、誰も知りません

   ```

   ```
   １、電車で行きます
   ２、何でここへきましたか
   ```

4. と表示共同做某事的对象，和

   ```
   家族と日本へ来ました
   ```

# 第八课



## 8.1 文型

1. 桜はきれいです
2. 富士山は高いです
3. 桜は綺麗い花です
4. 富士山はたかい山です



## 8.2 语法

1. 形容词的分类

   | 分类         | 例子                         |
   | ------------ | ---------------------------- |
   | い形容词     | いい、美味しい、古い、小さい |
   | な形容词     | ハンサム、簡単、元気         |
   | 特殊な形容词 | 綺麗い、嫌い                 |

2. 形容词做定语

   ```
   １、い形容词 + 名词
   サントスさんはいい人です。
   
   2、な形容词 + な + 名词
   京都は有名な町です
   ```

   

3. 形容词做谓语

   |      | い形容词               | な形容词               |
   | ---- | ---------------------- | ---------------------- |
   | 肯定 | 〜　＋　です           | 〜　＋　です　簡単です |
   | 否定 | 〜い　ー＞くない＋です | 〜　＋　ではありません |

   ```
   肯定
   1、安いです
   ２、簡単です
   
   否定
   １、安くないです
   ２、簡単ではありません
   
   
   注意：いい否定形式是形式是よくない　
   ```

   

4. どんな　什么样的

   ```
   奈良はどんな町ですか
   古いまちです
   ```

   

5. が　但是

   ```
   日本の食べ物は美味しいですが、高いです
   ```

   

6. そして：而且

   ```
   Tokyoの地下鉄は綺麗いです。そして、便利です。
   ```

   

7. どう：怎么样

   ```
   日本の生活はどうですか
   ```

   

8. とても・あまり　程度副词，修饰形容词时放在形容词前面

   ```
   1、 とても　+ 肯定
   北京はとても寒いです
   これはとても有名な映画です
   
   2、 あまり　+ 否定
   シャンハイはあまり寒くないです
   さくら大学はあまり有名な大学じゃありません
   ```

   

9. 哪个：从三件物品以上，选出一个特定的。

   ```
   ミラーさんの傘はどれですか
   ```



# 第九课



## 9.1 文型

1. 私はイタリア料理が好きです
2. 私は日本語が少しわかります
3. 今日は子供の誕生日ですから、早く帰ります。



## 9.2 语法

1. が表示情感、能力的对象

   ```
   私は歌が下手です
   ```

2. 副词

   ```
   １、よく
   ２、だいたい
   ３、たくさん
   ４、少し
   ５、あまり
   ６、全然
   
   
   11、表示程度
   よくわかります
   だいたいわかります
   少しわかります
   
   あまりわかりません
   全然わかりません
   
   12、表示数量
   たくさんあります
   少しあります
   あまりありません
   全然ありません
   ```

   

3. から表示因果关系

   ```
   時間がありませんから、新聞をの読みません
   ```

4. どうして询问原因。回答时句尾加から

   ```
   どうして新聞をの読みませんか
   時間がありませんから
   ```

5. どんな　＋　名词，询问某个范围内的具体名称

   ```
   どんなスポーツが好きですか
   野球が好きです
   ```

   



# 第十课

2. 机の上に写真があります
3. 家族はニューヨークにいます
4. ミラーさんは事務所にいます

## 10.2 语法

1. 存在句

   ```
   1、名词（无意志物）があります
   公園があります
   
   ２、名词（有意志物）がいます
   猫がいます
   ```

2. 某物在某个场所，用に提示

   + 某地有某物

     ```
     场所 に　名词 が　あります/います
     
     会議室に誰がいますか
     ```

   + 某物在某地

     ```
     名词 は　场所 に　あります/います
     
     ミラーさんは事務所にいます
     ```

     

3. 常用方位名词

   ```
   1、机の上に写真があります
   ２、ミラーさんは車の中にいます
   ３、郵便局は銀行とスーパーの間にあります
   ```

4. や列举有代表性的几个名词，名词1や名词2

   ```
   箱の中に手紙や写真があります
   ```

5. など：等等

   ```
   箱の中に手紙や写真などがあります
   ```



# 第十一课



## 11.1 文型

1. 会議室にテーブルが七つあります
2. 私は日本に一年います

## 11.2 语法

1. 数量词

2. 数字 + 量词。一般放在要修饰的动词前面

   ```
   りんごを三つ買いました
   外国人の学生が二人います
   ```

3. 针对数量词提问

   ```
   1、いくつ
   みかんをいくつ買いましたか
   八つ買いました
   
   ２、なん + 量词
   写真を何枚撮りましたか
   １００枚撮りました
   ```

4. 针对时间段提问

   ```
   １、どのぐらい
   どのぐらい日本語を勉強しましたか
   
   ２、なん＋年/か月/週間/日/時間/分
   日本語を何年勉強しましたか
   三年勉強しました
   ```

5. に表示一定时间内该动作的频率

   ```
   时间段 に 次数
   一週間に一回スーパーへ行きます
   ```

   

6. くらい/ぐらい　表示大约

   ```
   会社から家まで、タクシーで３０分ぐらいかかります
   ```

   

7. だけ表示只有这些

   ```
   昨日は２時間だけ寝ました
   ```

   



# 第十二课



## 12.1 文型

1. 昨日雨でした
2. 昨日は寒かったです
3. 北海道は九州より大きいです
4. 私は一年で夏が一番好きです

## 12.2 语法

1. 形容词和名词的过去式

   ```
   [い形容词]
   肯定    = 美味しいです
   否定    =美味しくないです
   过去肯定 = 美味しかったです
   过去否定 =美味しくなかったです
   
   [な形容词]
   肯定    = 静かです
   否定    =　静かではありません
   过去肯定 = 静かでした
   过去否定 =　静かではありませんでした
   
   [名词]
   肯定    = 雨です
   否定    =　雨ではありません
   过去肯定 = 雨でした
   过去否定 =　雨ではありませんでした
   ```

   

2. より　...比...

   ```
   この車はあの車より大きいです
   ```

3. 哪个更....

   ```
   名词1と名词2　と　どちらが形容词ですか
   テニスとサッカーと、どちらが面白いです
   ```

4. 选择程度最高的疑问句，用で表示范围

   ```｀｀｀｀
   一年でいつが一番寒いですか
   ```



# 第十三课



## 13.1 文型

1. 私はパソコンが欲しいです
2. 私は天ぷらが食べたいです
3. 日本語の勉強に来ました

## 13.2 语法

1. 想要

   ```

   ```


3. に表示移动行为的目的。去/来做....

   ```

   神戸へ買い物に行きます
   日本語の勉強にきました
   ```

4. 疑问词か表示不确定 某人/某地/某时

   ```
   誰/どこ/いつ/なに＋か
   夏休みどこか行きましたか
   ```

5. に提示进入的场所

   ```
   あの喫茶店に入りましょう
   ```

6. を提示离开的场所

   ```
   七時に家を出ます
   ```

# 第四课



## 4.1 文型

1. 今四時五分です
2. 私は九時から五時まで働きます
3. 私は朝六時に起きます
4. 私は昨日勉強しました



## 4.2 语法

1. 动词的分类

   + 一类动词：词尾不是る；ア、ウ、オ段 + る

     ```
     １、休む
     ２、働く
     ３、終わる
     ４、分かる
     
     
     特殊一类动词
     １、帰る
     ２、切る
     ３、知る
     ４、入る
     ５、いる
     ```

     

   + 二类动词：ｲ、エ段+る

     ```
     １、起きる
     ２、寝る
     ```

     

   + 三类动词：する、くる

     ```
     １、勉強する
     ２、来る
     ```

     

     

2. 动词句

   动词连用形 + ます

   |      | 现在、将来 | 过去         |
   | ---- | ---------- | ------------ |
   | 肯定 | ます       | ました       |
   | 否定 | ません     | ませんでした |

   

3. 时间表示

   ```ini
   １、小时
   4点 = よじ
   7点 = しちじ
   ９点 = くじ
   
   2、分钟
   1分 = 　いっぷん
   3分 =　さんぷん
   4分 = よんぷん
   6分 = ろっぷん
   8分 = 　はっぷん
   10分=　じゅっぷん
   30分= はん
   ```

   

4. 在某个时间做某事

   ```
   １、具体时间  に + 动词
   十一時に起き
   
   ２、非具体时间
   私は毎日働きます
   ```

   

5. から・・・まで・・・

   ```
   １、九時から五時まで、
   ２、十時から勉強します
   ３、九時まで寝ました
   ```

   

6. と：相当于和

   ```
   休みは土曜日と日曜日です
   ```

# 第八课



## 8.1 文型

1. 桜はきれいです
2. 富士山は高いです
3. 桜は綺麗な花です
4. 富士山はたかい山です



## 8.2 语法

1. 形容词的分类

   | 分类         | 例子                         |
   | ------------ | ---------------------------- |
   | い形容词     | いい、美味しい、古い、小さい |
   | な形容词     | ハンサム、簡単、元気         |
   | 特殊な形容词 | 綺麗、嫌い                   |

2. 形容词做定语

   ```
   １、い形容词 + 名词
   サントスさんはいい人です。
   
   2、な形容词 + な + 名词
   京都は有名な町です
   ```

   

3. 形容词做谓语

   |      | い形容词               | な形容词               |
   | ---- | ---------------------- | ---------------------- |
   | 肯定 | 〜　＋　です           | 〜　＋　です　簡単です |
   | 否定 | 〜い　ー＞くない＋です | 〜　＋　ではありません |

   ```
   肯定
   1、安いです
   ２、簡単です
   
   否定
   １、安くないです
   ２、簡単ではありません
   
   
   注意：いい否定形式是形式是よくない　
   ```

   

4. どんな　什么样的

   ```
   奈良はどんな町ですか
   古いまちです
   ```

   

5. が　但是

   ```
   日本の食べ物は美味しいですが、高いです
   ```

   

6. そして：而且

   ```
   東京の地下鉄は綺麗です。そして、便利です。
   ```

   

7. どう：怎么样

   ```
   日本の生活はどうですか
   ```

   

8. とても・あまり　程度副词，修饰形容词时放在形容词前面

   ```
   1、 とても　+ 肯定
   北京はとても寒いです
   これはとても有名な映画です
   
   2、 あまり　+ 否定
   シャンハイはあまり寒くないです
   さくら大学はあまり有名な大学じゃありません
   ```

   

9. 哪个：从三件物品以上，选出一个特定的。

   ```
   ミラーさんの傘はどれですか
   ```

## 12.2 语法

1. 形容词和名词的过去式

   ```
   [い形容词]
   肯定    = 美味しいです
   否定    =美味しくないです
   过去肯定 = 美味しかったです
   过去否定 =美味しくなかったです

   肯定    = 静かです
   否定    =　静かではありません
   过去肯定 = 静かでした
   过去否定 =　静かではありませんでした
   
   [名词]
   肯定    = 雨です
   否定    =　雨ではありません
   过去肯定 = 雨でした
   过去否定 =　雨ではありませんでした
   
   [动词]
   肯定    = 働きます
   否定    =　働きません
   过去肯定 = 働きました
   过去否定 =　働きませんでした
   ```

   

2. より　...比...

   ```
   北海道は九州より大きいです
   ```

3. 哪个更....

   ```
   名词1と名词2　と　どちらが形容词ですか

## 14.1 文型

1. ちょっと待ってください
2. ミラーさんは今電話をかけています

## 14.2 语法

1. 动词的て形

   + 一类动词

     ```
     １、く　ー＞　いて　ぐ　ー＞　いで
     書くー＞　かいて　泳ぐー＞　およいで
     
     ２、う、つ、る结尾→去掉う、つ、る+って
     もらう　ー＞　もらって　待つ　ー＞　まって　ある　ー＞　あって
     
     ３、む、ぬ、ぶ结尾→去掉む、ぬ、ぶ＋んで
     読む　ー＞　よんで　死ぬ　ー＞　しんで　遊ぶ　ー＞　あそんで
     
     ４、す结尾→して
     貸す　ー＞　かして
     
     ５、特殊 いく　ー＞　行って
     ```

     

   + 二类动词

     ```
     去掉 る　+　て
     食べる　ー＞　食べて
     ```

     

   + 三类动词

     ```
     する　ー＞　して
     くる　ー＞　きて
     ```

5. が表示轻微的转折

   雨の日は地面が滑りやすい。
   
   
   2、很难发生
   このコップは割れにくい。
   ```

3. 〜します：做成……，弄成……

   ```
   # 用法类似なる
   い形容词去掉い + く
   クーラーをつけて、部屋を涼しくします。
   
   な形容词 + に
   みなさん、静かにしてください。
   
   # 表示决定做某事
   名词 + に 
   晩ご飯はカレーにします。
   ```



# 第四十五课



## 45.1 文型

1. カードをなくした場合は、すぐカード会社に連絡してください。
2. 約束をしたのに、彼女はきませんでした。

## 45.2 语法

1. 〜場合は：……的时候，……的话

   ```
   普通型 + 場合は
   な形容词な
   名词の
   
   对比
   １・時
   ２・場合は：全部可以换成とき
   ３・たら
   
   # 过去的事实，不能用 場合は
   子供の時、よくこの公園で遊びました。
   
   会社を休む「時・場合」は、会社に連絡してください。
   
   もし生まれ変わったら、鳥になりたいです。
   部長が来たら、会議を始めはしょう。
   ```

2. 〜のに：……却，后半句描述的事情与前半句预想不同，表示意外和不满

   ```
   　　　な形容词、名词
   普通型(な・な)のに
   
   薬を飲んだのに、よくなりません。
   え？今日のテストは中止？昨日、寝ないで勉強したのに
   
   
   对比42课的のに
   これは缶を開けるのに使います。
   ```



# 第四十六课



## 46.1 文型

1. 会議は今から始まるところです。
2. 彼は３月に大学を卒業したばかりです。
3. ミラーさんは会議室にいるはずです。

## 46.2 语法

1. 〜ところです：

   ```
   1、 辞书形 + ところです：正要做……
   今から食べるところです
   
   2、て形いる　+ ところです 正在做……
   今食べているところです
   
   3、 た形　+ ところです 刚才做……
   たった今食べたところです。
   
   4、ています：可以替换 て形いる　+ ところです
   ```

2. 〜ばかりです：刚做……

   ```
   4月にこの会社に入ったばかりなので、まだ慣れません。
   
   先週買ったばかりなので、もう壊れてしまいました。
   ```

3. 〜はずです：应该……，照理说……

   ```
   　　　　な形容词 名词
   普通形（な・の）はずです
   ミラーさんは会議室にいるはずです。
   
   普通形+ でしょう 应该，客观的推断
   ```





# 第四十七课



## 47.1 文型

1. 天気予報によると、明日は寒くなるそうです。
2. 隣の部屋に誰かいるようです。

## 47.2 语法

1. 〜そうです：听说，完全不加自己的判断

   ```
   　　　　な形容词  名词
   普通形（だ・だ） + そうです
   
   # 听说要下雨
   雨が降るそうです。
   
   # 好像要下雨
   雨が降りそうです。
   ```

2. 〜ようです：好像……；像……一样

   ```
   1、表示推断
   父が帰ってきたようです。
   
   2、举例
   彼が持っているようなかばんがほしい。
   
   3、比喻
   彼女の肌は雪のように白い。
   
   ```

3. 补充

   ```
   1、47课：そうです
   听说，不加任何判断
   
   2、43课：そうです
   看起来，视觉的，直观的判断
   
   3、ようです
   五感的，基于逻辑的判断
   ```

# 第四十八课



## 48.1 文型

1. 息子をイギリスへ留学させます。
2. 娘にピアノを習わせます。

## 48.2 语法

1. 使役动词

   ```
   1、一类动词
   う段 改为 あ段 + せる
   いく　ーー　行かせる
   
   2、二类动词
   去掉る + させる
   食べる　ーー　食べさせる
   
   3、三类动词
   きる　ーー　こさせる
   する　ーー　させる
   ```

2. 使役动词句

   ```
   1、A　は　B　を　自动词使役形
   A叫B 做 某事
   私は娘を電車の中で立たせました。
   
   私は息子をプールで泳がせました。
   
   2、A　は　B　に　他动词使役形
   A叫B做某事，强制
   A让B做某事，许可
   
   # 强制
   私は息子に皿を洗わせます。
   
   # 许可
   私は娘に好きなんものを食べさせます。
   
   # 诱发
   私は彼女を怒らせてしまった。
   ```

3. 补充

   ```
   て形いただけませんか：能否请您做……
   
   使役て形いただけませんか：能否请您让我做……
   すみませんか、明日休ませていただけませんか。
   不好意思，明天能否让我请假
   
   
   A　は　B　に　他动词使役被动形
   A被B逼迫做某事
   
   私は母にピーマンを食べさせられました。
   
   食べる　ーー　食べさせる　ーー　食べさせられる
   ```

# 第一章 如何吃

## 训练前

+ 训练前不吃东西最好，比如早上空腹去训练，或者不吃午餐去训练
+ 训练前三小时吃一个正餐



## 训练后

训练后分两次吃

+ 训练后半小时，吃一个馒头、贝果等补充碳水，量一定要少，否则会导致胃不舒服
+ 训练后2小时吃正餐（蛋白质、碳水、蔬菜都要吃够）
+ 蛋白粉晚上睡觉前喝



# 第二章 膳食纤维

| 类别           | 可溶性膳食纤维                             | 不可溶性膳食纤维                       |
| -------------- | ------------------------------------------ | -------------------------------------- |
| 溶解性         | 可溶于水，形成凝胶状物质                   | 不溶于水，保持颗粒状                   |
| 作用机制       | 延缓食物消化，增加肠道益生菌，软化粪便     | 增加粪便体积，促进肠道蠕动，加快排便   |
| 对便秘的作用   | 吸水膨胀，使粪便湿润，帮助温和排便         | 增加粪便体积，刺激肠道，加快粪便通过   |
| 对血糖的影响   | 降低糖分吸收，帮助控制血糖                 | 影响较小                               |
| 对胆固醇的影响 | 能结合胆汁酸，降低胆固醇                   | 影响较小                               |
| 常见食物来源   | 燕麦、菊粉、果胶（苹果、柑橘）、豆类、坚果 | 全麦食品、糙米、芹菜、胡萝卜、坚果外皮 |

**改善便秘**：两者结合效果更好

**控制血糖或胆固醇**：优先摄入**可溶性膳食纤维**

**增加粪便体积，缓解排便困难**：优先摄入**不可溶性膳食纤维**



可溶性膳食纤维

### **1. 全谷类**

- **燕麦**（富含β-葡聚糖）
- **大麦**
- **黑麦**

### **2. 豆类**

- **黑豆**
- **红豆**
- **鹰嘴豆（鸡豆）**
- **扁豆**

### **3. 水果**

- **苹果**（含果胶）
- **柑橘类（橙子、柚子、柠檬）**
- **梨**
- **香蕉**（特别是未成熟的青香蕉，富含抗性淀粉）
- **鳄梨（牛油果）**

### **4. 蔬菜**

- **胡萝卜**
- **西兰花**
- **地瓜（红薯）**
- **南瓜**

### **5. 坚果和种子**

- **亚麻籽**
- **奇亚籽**
- **葵花籽**
- **罗勒籽**

### **6. 其他**

- **菊粉（Inulin）**（天然存在于菊苣根、洋葱、大蒜、韭菜等）
- **海藻类**（如海带、裙带菜，富含藻胶）

### **建议摄入方式**

- 早餐加**燕麦、奇亚籽**
- 多吃**水果和蔬菜**，尤其是连皮吃（如苹果、梨）
- 适量吃**豆类和坚果**，如做汤或拌沙拉

适量增加可溶性膳食纤维可以改善消化、促进益生菌生长、降低胆固醇并稳定血糖，但要**搭配足够水分**，以避免肠胃不适。



以下是**不可溶性膳食纤维**含量较高的食物：

### **1. 全谷类**

- **全麦面包、全麦面粉**
- **糙米、黑米**
- **燕麦麸、小麦麸**
- **玉米、小米、藜麦**

### **2. 豆类**

- **黑豆**
- **红豆**
- **绿豆**
- **黄豆**

### **3. 蔬菜**

- **芹菜**
- **胡萝卜**
- **菠菜、甘蓝、羽衣甘蓝**
- **西兰花**
- **白菜、卷心菜**

### **4. 水果（带皮食用效果更好）**

- **苹果（连皮）**
- **梨（连皮）**
- **葡萄（连皮）**
- **李子**
- **香蕉（尤其是较成熟的香蕉）**

### **5. 坚果和种子**

- **杏仁**
- **核桃**
- **葵花籽**
- **南瓜籽**

### **6. 其他**

- **薯类**（红薯、土豆，尤其是带皮食用）
- **蘑菇**（如香菇、白蘑菇）
- **竹笋**

### **如何更好地摄取？**

- **主食选择**糙米、全麦面包、玉米等粗粮
- **多吃深色叶菜**（如菠菜、西兰花）
- **水果尽量连皮吃**（苹果、梨、葡萄）
- **适量坚果和种子**，如杏仁、南瓜籽

### **作用与注意事项**

不可溶性膳食纤维能**增加粪便体积、促进肠道蠕动**，对防止便秘、保持肠道健康有帮助。但**如果摄入过多且饮水不足，可能会导致腹胀或便秘加重**，因此建议**搭配充足水分**。

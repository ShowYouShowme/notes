

# 终端

1. 使用颜色区分文件和目录

   ```shell
   # ~/.bash_profile
   export CLICOLOR=1
   export LSCOLORS=GxFxCxDxBxegedabagaced
   
   # source ~/.bash_profile
   ```



# 包管理器

1. 安装brew

   ```shell
   /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
   ```
   
2. 安装telnet

   ```shell
   brew install telnet
   ```



# 常用命令

1. 保存

   ```shell
   win + s
   ```

2. 复制
   ```shell
   alt + c
   ```
   
3. 粘贴
   ```
   alt + v
   ```
   
4. 撤销
   ```
   win + z
   ```
   
5. 剪切

   ```shell
   win + x
   ```
